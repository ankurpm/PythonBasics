✅ Beginner-Level Set Exercises
1. Create and Print a Set

Task:
Create a set called fruits with these items: "apple", "banana", "orange".
Print the set.

2. Add Items to a Set

Task:
Start with this set:

colors = {"red", "blue"}


Add "green" and "yellow" to it.
Print the updated set.

3. Remove an Item

Task:
Given:

animals = {"cat", "dog", "rabbit", "elephant"}


Remove "rabbit" from the set.
Print the result.

4. Check Membership

Task:
Ask the user for a color and check if it exists in:

{"red", "blue", "green", "yellow"}


Print "Found" or "Not Found".

✅ Intermediate-Level Set Exercises
5. Set Operations: Union

Task:
Given:

a = {1, 2, 3}
b = {3, 4, 5}


Find the union of both sets.

6. Set Intersection

Task:
Find the common elements between:

x = {"apple", "mango", "banana"}
y = {"banana", "kiwi", "apple"}

7. Difference of Sets

Task:
Using:

set1 = {10, 20, 30, 40}
set2 = {30, 40, 50}


Find elements that are in set1 but not in set2.

8. Symmetric Difference

Task:
Find items present in either of these sets but not in both:

p = {1, 2, 3}
q = {3, 4, 5}

9. Convert List to Set

Task:
Remove duplicates from:

nums = [1, 2, 2, 3, 4, 4, 5]


Print the result as a set and also as a list without duplicates.

✅ Advanced-Level Set Exercises
10. Unique Words Counter

Task:
Ask the user to enter a sentence.
Count how many unique words it has using a set.

11. Students in Sports

Task:
You have:

cricket = {"John", "Ali", "Priya", "Sara"}
football = {"Ali", "George", "Priya", "Michael"}


Find:

Students who play both sports

Students who play only cricket

Students who play only football

Students who play at least one sport

12. Check if Two Strings Are Anagrams Using Sets

Task:
Write a program to check if two input strings are anagrams (ignore spaces and case) using sets and counts.

13. All Characters Unique?

Task:
Write a function that checks if a string has all unique characters using a set.

Example:
"hello" → False
"abc" → True

14. Find Duplicate Items in a List

Task:
Given:

items = ["pen", "pencil", "pen", "marker", "pencil", "eraser"]


Use sets to print:

Unique items

Duplicate items

15. Frozenset Exercise

Task:
Create a frozenset from this list:

[10, 20, 20, 30]


Try adding a new value.
Explain why Python throws an error.
